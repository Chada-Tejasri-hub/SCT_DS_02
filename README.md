Titanic Dataset - Exploratory Data Analysis (EDA)

Project Overview

This project focuses on Data Cleaning and Exploratory Data Analysis (EDA) of the Titanic Dataset using Google Sheets. The objective is to identify patterns, trends, and factors that influenced passenger survival.

Dataset Information

- Dataset: Titanic Dataset
- Total Passengers: 891
- Features Analyzed: Passenger Class, Gender, Age, Fare, Embarked Port, and Survival Status

Tools Used

- Google Sheets
- Pivot Tables
- Charts and Graphs
- Data Cleaning Techniques

Data Cleaning Performed

- Filled missing Age values using median.
- Filled missing Embarked values using mode.
- Removed/ignored Cabin column due to excessive missing values.
- Checked and removed duplicate records.

Analysis Performed

Survival Rate

- Survived: 342 (38.38%)
- Not Survived: 549 (61.62%)

Gender Analysis

- Female Survival Rate: 74.20%
- Male Survival Rate: 18.89%

Passenger Class Analysis

- 1st Class Survival Rate: 62.96%
- 2nd Class Survival Rate: 47.28%
- 3rd Class Survival Rate: 24.24%

Embarkation Port Analysis

- Southampton: 33.70%
- Cherbourg: 55.36%
- Queenstown: 38.96%

Age Group Analysis

- 0–18 Years: 52.60%
- 19–35 Years: 36.35%
- 36–60 Years: 41.38%
- 60+ Years: 22.22%

Dashboard

The dashboard contains:

- EDA Summary Table
- Survival Rate Pie Chart
- Gender-Based Survival Analysis
- Passenger Class Analysis
- Embarkation Port Analysis
- Age Group Analysis
- Key Insights
- Data Cleaning Summary

Key Insights

- Overall survival rate was 38.38%.
- Female passengers had significantly higher survival rates than males.
- First-class passengers had the highest survival probability.
- Passengers embarking from Cherbourg had better survival rates.
- Children had better survival chances compared to older passengers.

Conclusion

The analysis revealed that Gender, Passenger Class, and Age were the most influential factors affecting survival on the Titanic. Female passengers and first-class travelers had the highest likelihood of survival.
